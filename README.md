# 📖 Random Greeting <a name="about-project"></a>

The **random greeting app** demonstrates using ruby on rails with webpacker and reactJs. 
When a user visits the homepage they get a different greeting whenever the page reloads.

## 🛠 Built With <a name="built-with"></a>

Ruby on Rails, Postgresql,ReactJs, RubyMine-IntelliJ

## 💻 Getting Started <a name="getting-started"></a>

### Setup <a name="setup"></a>

In order to use this project.. Clone this repository to your desired folder by:

- git clone https://github.com/NoelLincoln/random-greeting.git

### Prerequisites <a name="prerequisites"></a>

- Ruby
- Ruby on Rails
- ReactJs
- PostgreSQL
- Git

### Install <a name="install"></a>

To install the project's dependencies, run:

```
bundle install
```

```
npm install
```

### Usage <a name="usage"></a>

To use, run:

```
./bin/dev
```
